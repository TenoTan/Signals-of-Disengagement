# Signals of Disengagement  
### Jaya Optimization-Based Clustering for User Engagement & Feature Abandonment Analysis

## Overview
This project focuses on identifying early signals of user disengagement and feature abandonment in digital platforms using advanced data mining techniques. It models user behavior as a **temporal analysis problem**, capturing how engagement evolves over time rather than treating it as a static classification task.

The system is designed to help organizations understand whether newly introduced features are actually useful, underutilized, or contributing to user drop-off.

---

## Problem Statement
Modern digital platforms continuously roll out new features, but lack clear visibility into:
- Whether users are actually adopting these features
- When and why users start disengaging
- How feature usage impacts customer retention

Traditional models often provide only final predictions (e.g., churn/no churn) without explaining **behavioral progression**.

---

## Proposed Solution
This project introduces a **hybrid clustering-based framework** that:

- Segments users based on engagement behavior
- Detects early-stage disengagement signals
- Tracks feature adoption and abandonment patterns
- Identifies stages of service discontinuation

---

## Key Techniques Used

### 1. Jaya Optimization-Based Clustering
- Optimizes cluster centroids instead of relying on random initialization
- Avoids local minima issues seen in traditional clustering (e.g., K-means)
- Parameter-free optimization approach

### 2. Exploration–Exploitation Enhancement
Inspired by modern metaheuristic strategies:
- Adaptive control mechanisms (dynamic balance between search & refinement)
- Diversity preservation techniques
- Optional perturbation strategies (to avoid premature convergence)

### 3. Temporal Behavioral Analysis
- Incorporates time-based features such as:
  - Session duration
  - Click frequency
  - Feature usage trends
- Enables tracking of engagement decline over time

## Unique Contributions

- **Behavioral Progression Modeling**  
  Goes beyond static churn prediction by identifying *stages of disengagement*

- **Feature-Level Insights**  
  Detects which features are:

- **Decision Support for Product Teams**  
  Helps companies:
  - Decide whether to improve or discontinue features
  - Identify features causing user drop-off
  - Align product development with actual user behavior

- **Early Warning System**  
  Identifies users at risk *before* complete disengagement

- **Hybrid Optimization Approach**  
  Combines:
  - Global search capability (exploration)
  - Local refinement (exploitation)
  for superior clustering performance
